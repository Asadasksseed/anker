# Anker Dev — افزونهٔ ووکامرس

افزونهٔ **Anker Dev** ساختهٔ [shabake.dev](https://shabake.dev/) مجموعه‌ای از ویژگی‌های توسعه‌دهنده برای ووکامرس را در یک رابط مدیریتی واحد و با طراحی نیتیو وردپرس فراهم می‌کند.

> سازگار با ووکامرس **۱۰.۷+** و وردپرس **۶.۵+** (تست‌شده تا **وردپرس ۷.۰** و ووکامرس ۱۰.۷).
> پشتیبانی رسمی از HPOS (جدول‌های سفارش سفارشی).

## ساختار مخزن

```
anker-dev/                              ← خود افزونهٔ وردپرس
├── anker-dev.php                       ← فایل اصلی افزونه (هدر + بوت‌استرپ)
├── readme.txt                          ← فایل readme استاندارد وردپرس
├── uninstall.php                       ← پاک‌سازی هنگام حذف افزونه
├── assets/
│   └── css/admin.css                   ← استایل‌های نیتیو پنل مدیریت
├── languages/
│   ├── anker-dev-fa_IR.po
│   └── anker-dev-fa_IR.mo
└── includes/
    ├── class-anker-dev-plugin.php      ← Bootstrap (singleton)
    ├── class-anker-dev-settings.php    ← مدیریت گزینه‌ها
    ├── class-anker-dev-admin.php       ← منو و صفحهٔ ویژگی‌ها
    ├── abstract-anker-dev-feature.php  ← کلاس پایهٔ ویژگی‌ها
    └── features/
        └── class-anker-dev-cancel-pending-orders.php
```

## ویژگی‌های فعلی

### لغو خودکار سفارش‌های پرداخت‌نشده

اگر سفارشی در وضعیت «در انتظار پرداخت» ثبت شود و در مدت تعیین‌شده پرداخت نشود، به‌صورت خودکار لغو می‌شود.

- مدت زمان قابل تنظیم بین ۱ تا ۱۴۴۰ دقیقه (پیش‌فرض ۱۵ دقیقه).
- قابلیت فعال/غیرفعال‌سازی مستقل از تنظیمات نگهداری موجودی ووکامرس.
- پیاده‌سازی با Action Scheduler (همان زیرساختی که خود ووکامرس برای کارهای زمان‌بندی‌شده استفاده می‌کند).
- یک Sweep پشتیبان هر ۵ دقیقه که سفارش‌های جامانده را نیز پاک‌سازی می‌کند.
- یادداشت فارسی روی سفارش لغوشده ثبت می‌شود.

## نصب

1. پوشهٔ `anker-dev/` را در `wp-content/plugins/` بارگذاری کنید.
2. از منوی «افزونه‌ها» آن را فعال کنید.
3. به منوی **انکر دِو** در پیشخوان بروید و ویژگی‌ها را پیکربندی کنید.
