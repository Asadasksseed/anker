مجموعه‌ای از ویژگی‌های توسعه‌دهنده برای ووکامرس با رابط کاربری فارسی و طراحی نیتیو وردپرس.

== Description ==

افزونهٔ **Anker Dev** ساختهٔ [shabake.dev](https://shabake.dev/) مجموعه‌ای از ویژگی‌های توسعه‌دهنده برای ووکامرس است که در یک صفحهٔ مدیریتی واحد و با طراحی نیتیو وردپرس قابل مدیریت‌اند.

= ویژگی‌های نسخهٔ ۱.۰.۰ =

* **لغو خودکار سفارش‌های پرداخت‌نشده** — اگر سفارشی در وضعیت «در انتظار پرداخت» ثبت شود و در مدت زمان مشخص (پیش‌فرض ۱۵ دقیقه) پرداخت نشود، به‌صورت خودکار لغو می‌شود.
  * قابلیت فعال یا غیرفعال‌سازی مستقل
  * قابلیت تنظیم مدت زمان (بین ۱ تا ۱۴۴۰ دقیقه)
  * استفاده از Action Scheduler ووکامرس برای زمان‌بندی دقیق
  * سازگار با HPOS (جدول‌های سفارش سفارشی)
  * یک یادداشت فارسی به سفارش لغوشده افزوده می‌شود

ویژگی‌های بیشتر در نسخه‌های آینده اضافه خواهند شد.

== Installation ==

1. پوشهٔ `anker-dev` را در مسیر `/wp-content/plugins/` آپلود کنید (یا فایل ZIP را از طریق صفحهٔ افزونه‌های وردپرس بارگذاری کنید).
2. افزونه را از منوی «افزونه‌ها» فعال کنید.
3. به منوی «انکر دِو» در پیشخوان وردپرس بروید و ویژگی‌ها را پیکربندی کنید.

پیش‌نیاز: ووکامرس نسخهٔ ۸.۰ یا بالاتر باید نصب و فعال باشد.

== Frequently Asked Questions ==

= آیا با HPOS سازگار است؟ =

بله. افزونه به‌صورت رسمی سازگاری خود را با `custom_order_tables` اعلام می‌کند.

= آیا می‌توانم به‌جای ۱۵ دقیقه، زمان دیگری تعیین کنم؟ =

بله. در صفحهٔ «انکر دِو» می‌توانید مقدار را بین ۱ تا ۱۴۴۰ دقیقه (یک شبانه‌روز) تنظیم کنید.

= آیا این ویژگی روی تنظیمات نگهداری موجودی ووکامرس تأثیر می‌گذارد؟ =

خیر. مکانیزم لغو در این افزونه کاملاً مستقل از گزینهٔ `woocommerce_hold_stock_minutes` ووکامرس عمل می‌کند.
